# example-repo
This is a programme to create an investment calculator as well as a bond repayment calculator using simple or compound interest.
Run the file financial_calculator_new.py in coding programe the user will first select between investment or bond. If investment is chosen, the user should input the amount where after the user must put in the interest rate and years of investment. the user must then select simple or compund interest and then will get the appropriate output. If the user selects bond repayment the user has to enter the value of the house then the interest rate of the bond and over how many months the bond is going to have to be repaid ,which gives the user the monthly instalement.
